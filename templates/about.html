À Propos de ScolarPredict

ScolarPredict est un système intelligent conçu pour la prédiction de la note du troisième trimestre (G3) et l'identification précoce des étudiants à risque. Ce projet s'inscrit dans le cadre d'une certification en Intelligence Artificielle et Data Science, visant à démontrer l'application concrète des techniques de Machine Learning pour améliorer le suivi pédagogique et la réussite scolaire.

Description Détaillée de l'Application

ScolarPredict offre une interface conviviale permettant aux utilisateurs (éducateurs, parents) de saisir des informations clés concernant un étudiant. Ces données sont ensuite traitées par un modèle de Machine Learning entraîné pour estimer la note finale du troisième trimestre. Au-delà de la simple prédiction numérique, l'application fournit une interprétation qualitative du résultat, signalant si l'étudiant est sur la bonne voie, s'il a de solides acquis, ou s'il nécessite un suivi pédagogique particulier pour éviter le décrochage. L'objectif est de transformer les données en informations exploitables pour une intervention rapide et ciblée.

Historique du Projet

Le projet ScolarPredict a débuté par une phase d'exploration des données académiques disponibles, suivie d'une étape cruciale de prétraitement et d'ingénierie des fonctionnalités. Plusieurs algorithmes de Machine Learning ont été expérimentés et évalués pour identifier le modèle le plus performant et le plus robuste pour la tâche de prédiction. L'intégration du modèle dans une application web Flask a permis de rendre cette solution accessible et interactive. Le développement a mis l'accent sur la clarté de l'interface utilisateur (grâce à Tailwind CSS) et la fiabilité des prédictions, tout en assurant une interprétation pédagogique des résultats.

Notre Technologie

L'application est construite avec le framework web Flask pour le backend, et utilise une interface utilisateur moderne conçue avec Tailwind CSS. Le cœur de notre service est un modèle de prédiction entraîné sur un ensemble de données d'étudiants, capable d'estimer la note finale (G3) avec une précision significative.

Comment les prédictions sont-elles faites ?

Le modèle prend en compte plusieurs facteurs, notamment :

  • Le temps d'étude hebdomadaire.
  • L'aspiration à poursuivre des études supérieures.
  • L'accès à Internet.
  • Le niveau d'éducation des parents.
  • Les notes obtenues lors des trimestres précédents (G1 et G2).

Ces informations nous permettent de dresser un profil de l'étudiant et de prédire sa trajectoire académique, tout en fournissant des interprétations pour guider les actions pédagogiques.

Ce projet est à but non lucratif et purement démonstratif.
